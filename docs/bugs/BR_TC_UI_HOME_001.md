- **TC_UI_HOME_001**

## Description
One of section headlines on the Automation Exercise homepage contains a typographical error. The displayed text in the *Featured Items* section does not match the expected wording defined in the related test case.

1. Open the Automation Exercise homepage.
2. Scroll to the *Featured Items* section.
3. Observe the section headline text.

The section headline should read **“Featured Items”**.

## Actual Result
The section headline displays **“Features Items”**.

- Automation Exercise website
